# Knowledge Base — Opinion Market (ex-Blind Bets)

## О проекте

**Opinion Market** — платформа для слепых ставок на вопросы-мнения. Пользователи ставят на одну из двух сторон, не видя текущих объёмов. Маркет разделён на блоки — в каждом блоке рандомная отсечка (Chainlink VRF), 30% банка выплачивается победившей стороне, 70% переносится дальше. После отсечки баланс блока раскрывается. Финальный блок — 100% остатка.

**Ключевые свойства:**
- Fully blind — объёмы скрыты до закрытия (ZK deposit proof)
- User-generated маркеты
- 4% комиссия (50% платформа / 25% создатель маркета / 25% рефералы)
- Chainlink VRF для рандомных отсечек

**Стек:** Monad L2, ZK proofs, Chainlink VRF, USDC, React/Next.js

## Решения (из встречи 06.03)
- Формат: 1 раунд по 8/16 часов
- Техническое решение: ZK депозиты
- Сеть: Monad L2
- Целевая аудитория: первая гипотеза — Верса (проверить Азуро/Трендл/букмекер)
